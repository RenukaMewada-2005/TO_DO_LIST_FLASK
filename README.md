# Flask To-Do List Web App

A simple To-Do List web application built using Flask.  
This app allows users to register, log in, and manage their daily tasks efficiently.

## Features
- User Registration and Login
- Add new tasks
- Update task status (Pending → Working → Done)
- Task deletion feature
- User-specific tasks
- Password hashing
- Clear all tasks
- Flash messages for user actions
- Clean and simple user interface

## Tech Stack
- Python
- Flask
- SQLAlchemy
- SQLite
- HTML
- CSS
